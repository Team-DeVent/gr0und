# gr0und-front
그라운드, 독립 메타버스 플랫폼. 웹 기반 low-level 메타버스 프로젝트

## Goal
그라운드 프로젝트는 메타버스 시스템의 중앙 집중화를 해결하기 위해 개발되었습니다. 다른 플랫폼처럼 설치형이 아닌 웹 기반으로 누구나 손쉽게 이용할 수 있다는 장점이 있습니다.
